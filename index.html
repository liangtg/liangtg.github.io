ubuntu16.04 ssh 添加两步验证

Thu, Jul 18, 2019 ubuntu16.04 ssh 添加两步验证 1 密码登陆 安装: apt install libpam-google-authenticator 运行: google-authenticator /etc/pam.d/sshd 第一行添加 auth sufficient pam_google_authenticator.so /etc/ssh/sshd 替换 ChallengeResponseAuthentication yes 重启 sudo service sshd restart 账号密码登陆时需要输入验证码 2 配置密钥登陆时也输入验证码(密码登陆时,不需要密码,需要验证码) /etc/ssh/sshd_config文件最后添加 AuthenticationMethods keyboard-interactive 参考 https://www.wevg.org/archives/ssh-add-mfa/ https://www.felix021.com/blog/read.php?2133 https://www.felix021.com/blog/read.php?2172
